2. 로직 판단을 위한 상세 체크리스트

※ 각 Step별로 하나라도 'YES'가 나오면 해당 속성을 가진 것으로 간주합니다.

[Step 0] 환경 평가 (Environment Check)

목적: 콘텐츠가 담긴 '그릇(블로그)'이 광고판인지 확인

구분 체크포인트 (Y/N 판단 기준) 관련 근거
채널 정체성 □ 블로그/채널명, 닉네임, ID에 의료기관 명칭(예: OO치과)이 포함되어 있는가?
구조적 노출 □ 블로그 스킨(PC 배경), 모바일 홈 화면, 프로필 영역에 전화번호, 진료시간, 주소가 기재되어 있는가?
위젯/배너 □ 사이드바나 하단에 '상담 문의', '예약하기', '카카오톡 친구추가' 등의 위젯/배너가 설치되어 있는가?
인접성 □ 포스트 목록이나 카테고리에 '진료 후기', '병원 이벤트', '공지사항' 등 명백한 병원 홍보 게시판이 노출되는가?
👉 판정: 하나라도 YES면 → '의료광고 환경' (Logic C 진입)

[Step 1] 콘텐츠 평가 (Content Check)

목적: 본문 내용이 '정보'를 가장한 '광고'인지 확인 (환경이 깨끗할 경우 진행)

구분 체크포인트 (Y/N 판단 기준) 관련 근거
식별성
(Identification)		 □ 본문 내 텍스트나 이미지로 의료기관 명칭(로고 포함)이 노출되는가?
□ 본문 하단에 네이버 지도(플레이스), 약도, 주소가 첨부되어 있는가?
□ 전화번호나 홈페이지/SNS 링크가 포함되어 있는가?
유인성
(Enticement)		 □ "저희 병원에서는", "제가 치료한" 등 1인칭 시점(주관적)으로 서술되었는가?
□ "내원하시면", "상담받아보세요" 등 행동 유도(Call to Action) 문구가 있는가?
□ 가격 할인, 이벤트, 무료 검진 등의 혜택이 언급되었는가?

구체성
(Specificity)		 □ 특정 환자의 치료 전후 사진이나 엑스레이가 사용되었는가?
□ 환자의 자필 후기, 인터뷰, 리뷰 캡처 이미지가 포함되었는가?
□ "본원에서 실제 치료한 사례입니다"라는 설명이 있는가?
👉 판정: 하나라도 YES면 → '광고성 콘텐츠' (Logic C 진입 가능성 높음)
👉 판정: 모두 NO이고, 환경 평가도 깨끗하다면 → Logic A (순수 정보)

[Step 2] 매체 및 면제 여부 평가 (Exemption Check)

목적: 광고임이 확인된 경우, 심의를 받아야 하는지 확인

구분 체크포인트 (Y/N 판단 기준) 관련 근거
매체 기준 □ 게시하려는 곳이 네이버 블로그/카페, 유튜브, 인스타그램, 페이스북인가?
(전년도 말 기준 직전 3개월 일일 평균 이용자 10만 명 이상)
단순 정보 □ 콘텐츠 내용이 오직 다음 항목들로만 구성되어 있는가?
- 의료기관 명칭, 소재지, 전화번호
- 진료과목 (법정 진료과목만)
- 의료인 성명, 성별, 면허 종류
- 진료일 및 진료시간, 홈페이지 주소
(수술법 설명, 장비 자랑, 인사말 등 일체 없음)
👉 판정: 매체 기준이 YES이고, 단순 정보가 NO라면 → Logic C (심의 필수)
👉 판정: 단순 정보가 YES라면 → Logic B (심의 면제)

3. 로직별 합법성 검토 기준 (Compliance Check)

아래 배지를 클릭하여 각 로직의 상세 검토 내용을 확인하세요.

Logic A (순수 정보) Logic B (심의 면제) Logic C (심의 대상)
🟢 Logic A: 순수 정보 (심의 X) 규제 대상 아님

[핵심] 의료광고가 아니므로 심의는 필요 없으나, 내용상 '불법 행위'가 포함되면 처벌받습니다.

구분위법 판단 기준근거
광고성 변질 □ 정보 제공 후 하단에 병원 지도, 상담 링크, 전화번호 노출 시 즉시 '의료광고'로 간주됨
타 의료인 비방 □ "다른 치과는 과잉진료하지만..." 등 경쟁자를 깎아내리는 내용
🟡 Logic B: 심의 면제 광고 (심의 X) 형식적 규제 면제

[핵심] 심의를 받을 필요는 없으나, 기재된 정보가 거짓이거나 과장되면 처벌받습니다.

구분위법 판단 기준근거
허위 정보 □ 전문의가 아님에도 'OO 전문의' 표기, 법적 진료과목 외 명칭 사용
비공인 인증 □ '소비자만족대상' 등 사설 상장 기재
범위 초과 □ "아프지 않은", "최고의 시설" 등 수식어 사용 시 Logic C로 전환됨
🔴 Logic C: 심의 대상 광고 (심의 O) 모든 규제 적용

[핵심] 반드시 사전 심의(절차)를 통과해야 하며, 통과했더라도 금지 내용이 포함되면 안 됩니다.

구분위법 판단 기준 (체크리스트)근거
1. 절차 위반 □ 유효한 '의료광고심의필 번호' 누락 (미심의 광고)
2. 환자 안전 □ 수술 장면 노출: 피, 절개된 잇몸, 뼈 노출 (혐오감 유발)
□ 미평가 신기술: 복지부 미인증 시술 광고
□ 치료보장: "100% 성공", "재발 없음" 등 단정적 표현
3. 소비자 오인 □ 전후사진 위반: 촬영시기, 부작용, 동일조건 미명시
□ 치료후기 공개: 로그인 없이 환자 자필 후기 노출
□ 최상급 표현: "최고", "유일", "완벽" 등 사용
4. 공정 경쟁 □ 사설 상장: 민간단체/언론사 상장 게시
□ 할인 유인: 기간/대상/범위 누락, "선착순" 유도

[판단] 네이버 블로그 등 주요 매체에서 가격 마케팅을 하려면 반드시 사전 심의를 통과해야 함. 		위법

Case 4. 수술 과정 상세 (File: 4.jpg)
🚨 위법 (Fatal)
🔗 원문 포스트 확인하기 (새창)

1. 시술행위 노출 금지
(치명적 위반)		 [분석] 잇몸을 절개하여 젖히고(Flap), 뼈와 임플란트 나사가 노출된 상태에서 피가 묻은 수술 도구가 삽입된 장면을 컬러 사진으로 게재함.
